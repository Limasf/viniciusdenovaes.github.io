## Comandos

Alguns comandos usados em aula para criar um ambiente virtual com python3, numpy e jupyter

virtualenv --python=python3 venv
source venv/bin/activate

apt-get update
apt-get install python3-virtualenv

pip install numpy
pip install jupyter
